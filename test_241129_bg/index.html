이미지의 크기조절

이미지 태그의 코딩한 이미지는 넓이나 높이 하나만 지정해주면 나머지는 비율에 맞게 리사이즈됨 (이미지 크기조정할때 넓이/높이 동시에 주면안됨) 이미지의 크기를 px 또는 % 등으로 조정할 수 있는데, %는 이미지를 감싸는 요소의 넓이를 기준으로 계산함 (원래 이미지크기와 상관없음)

background-size:직접 입력

배경의 크기를 px또는 %로 직접 입력할 수도 있습니다. 이때는 img 태그와 동일하게 넓이/높이 하나만 입력하면 비율에 맞게 사이즈가 줄어듭니다. (절대동시에 값을 주면안됨) background-size: 100px; - 넓이가 조절됨 (background-size: 100px auto; 이게 정확한 표현)
background-size: auto 100px - 높이가 100px로 조절되며 넓이가 비율에 맞게 조정됨 만약 단위를 %로 주면 배경을 준 요소의 넓이/높이를 기준을 % 계산함
