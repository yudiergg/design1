구글 아이콘, 폰트어썸 등에서 추천하는 방식

폰트 어썸은 굉장히 많이 사용되었음 (유료화되어서 지금안씀) si업체 솔루션 개발이나 어플리케이션 개발에서 많이 활용됨 (현재인기가 없음) 아이콘 폰트를 만들어서 코드를 입력하면 아이콘이 나타남 가벼움.. 용량이작음 경량화

  • 1. 구글 아이콘 홈페이지에서 아이콘 세팅하고 두께나 스타일 아무거나 클릭 (오른쪽에 코드가 나옴)
  • 2. 오른쪽 패널에서 static icon font 하단에 있는 css 링크 주소를 복사해서 넣어줌 (단 한번만 넣어줌)
  • 3. 실제 홈페이지에서 사용할 아이콘을 찾는다.
  • 4. 그 아이콘의 Inserting the icon 하단의 코드를 복사한다.
  • 5. 그걸 아이콘이 들어갈 위치에 붙여넣음
  • 6. span태그는 i태그로 변경. 절대 class명 수정 x 영문도 함부로 바꾸면안됨 그걸바꾸면 아이콘이 바뀜
  • 7. css에서 크기와 색상 변경 가능
